# Spis zadań powtórzeniowych
Wszystkie zadania z biorę z kurczaba, oznaczenia:
* `SR<numer klasy>` - stare rozszerzenie
* `NR<number klasy>` - nowe rozszerzenie
* `m<rok>z<zadanie>` - matura rozszerzona
